Project Overview

I had the opportunity to design the official website for PIMWork and also undertake a comprehensive redesign of their PIM (Product Information Management) software. This project involved creating a modern, user-friendly interface for their website and revamping the software to improve usability, efficiency, and overall user experience.

Key Contributions

  • Website Design: Crafted a clean, intuitive, and responsive design that effectively showcases PIMWork’s offerings, ensuring a seamless experience across devices.
  • Software Redesign: Reimagined the PIM software interface with a focus on enhancing user workflows, simplifying navigation, and optimizing functionality. The redesign prioritized user-centric design principles to create a more accessible and efficient tool for managing product information.
  • Marketing Design: Collaborated with the marketing team to design banners and social media content, helping to boost the brand's online presence and engagement.

Tools & Technologies

  • Figma for UI design and prototyping
  • Figjam and Miro for user flow and wireframing
  • Adobe Photoshop and Canva for marketing stuff

Impact

The redesigned website and software have been instrumental in enhancing the user experience, driving engagement, and improving overall satisfaction among PIMWork’s clients.
